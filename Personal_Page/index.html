Hi! My name is Gunjan Tomer and I am a graduate student in Computer Science at the University of Texas at Dallas. I am specializing in Intelligent Systems and working on various projects in Machine Learning and Aritficial Intelligence.

PROJECTS

MedBot - HackUTD 2017

Designed a conversational bot with a Machine Learning backend to diagnose different diseases based on the symptoms input by the user in FB chat. Used the wit.ai API to handle responses and Naive Bayes and Hidden Markov Models to make accurate predictions of the diseases.

Mining the Water Table - DrivenData Challenge

Made predictions on the operational status of hand pumps across Tanzania based on data provided by the Tanzanian Ministry for a data competition organized by DrivenData. Used various ML models like Random Forests, Gradient Boosting, Linear SVC, Logistic Regression to compare accuracies and retain the best prediction results with the help of scikit-learn and pandas libraries in Python.

Android App Development

Minimalistic Weather App

Made a weather app using the OpenWeatherMaps API to fetch localized weather data. Used various REST API features in a minimalistic, intuitive interface. Used the FusedLocationProviderClient API to monitor user location, also allowing manual locations and location history for weather updates including temperature, humidity, and pressure in different scales.

Contacts Manager

Designed a personalized contact manager app with options to add, edit and delete contacts, backup contacts to external storage or to Google Drive.

Implementation of Advanced algorithms and Data Structures

Implemented complex algorithms for flow problems, tour problems, complex data structures and a custom version of the Big Integer class in Java from scratch for the Implementation of Advanced Algorithms and Data Structures course. Used JUnit for unit testing on IntelliJ and Eclipse platforms.

Machine Learning Models and Spam/Ham Classifier

Coded various Machine Learning models from scratch as part of Machine Learning coursework at UT Dallas. Implemented Decision Tree, Logistic Regression in Java and a Naive Bayes Classifier in Python. Used the Naive Bayes classifier to build a spam/ham classifier which performed with an accuracy of 84% on real world data.

Database Design and Web Programming Project

Developed an efficient Airport Management System including backend design using MySQL, MAMP, PHP and frontend design using HTML5, AngularJS and CSS. Implemented CRUD operations on the flight database using a fully functional webpage.

Three-dimensional Scanner using Computer Vision

Designed a three-dimensional scanner capable of creating accurate scans of objects using the resolution of shadows from a structured light source. Used a simple laser and a custom frame to create shadows of the object and a high-resolution camera to record a sequence of images followed by processing using the Camera calibration and the Image Processing Toolbox in MATLAB. Used the ICP(Iterative Closest Point) algorithm to create a watertight mesh from the scans ready for 3-D printing.
